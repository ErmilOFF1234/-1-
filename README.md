# CRM для турагентства на 1С:Предприятие 8.3

## О проекте
Учебный проект, созданный в рамках самостоятельного изучения платформы 1С. 
CRM предназначена для автоматизации учёта турагентства.

## Функциональность
- Справочники: Клиенты, Туроператоры, Туры, Валюты
- Документы: Заявка на тур, Оплата туроператору
- Регистры: Расчёты с персоналом, Комиссия
- Отчёты: Продажи по турам, Доходы по комиссии

## Технологии
- Платформа 1С:Предприятие 8.3
- Управляемые формы
- Язык запросов 1С
- Система компоновки данных (СКД)

## Как открыть
1. Скачай файл `src/CRM_Turagency.cf`
2. В Конфигураторе 1С создай новую информационную базу
3. Выполни «Конфигурация» → «Загрузить конфигурацию из файла»
4. Выбери скачанный `.cf` файл
